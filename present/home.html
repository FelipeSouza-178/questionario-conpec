Passo-a-passo

Para realizarmos o projeto, a primeira etapa a se passar é pelo protótipo, na qual faremos o visual, então partimos para o desenvolvimento. Cada uma dessas etapas é descompromissada e não necessariamente precisam ser feitas com a Conpec.

Protótipo

No protótipo, faremos toda o design visual do projeto, seguindo todos os seus pedidos. É necessário passar por esta etapa, antes da programação, para poder remover qualquer empecilho e auxilar os desenvolvedores

Desenvolvimento

Com tecnologias como React Native, HTML/CSS e Firebase, desenvolvemos o seu projeto com base no protótipo feito, buscando agregar o máximo de valor possível ao projeto como um todo
